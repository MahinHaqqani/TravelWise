# TravelPlannerWebApp


This project was developed as part of the **AI Careers for Women program by WeHub and Microsoft**.

Travel planning often requires users to search multiple websites for destinations, read numerous reviews, check weather conditions, and organize travel schedules. Most existing travel platforms provide static recommendations and display reviews as plain text, making it difficult for travelers to quickly understand the overall opinion about a destination.

To address this issue, this project introduces **Travel Wise**, an AI-powered travel planner integrated with **review-based sentiment analysis** that generates personalized travel itineraries based on user preferences such as interests, destination, and travel duration.

The system is developed using **Flask** for the backend and **HTML, CSS, and JavaScript** for the frontend. Sentiment analysis is implemented using the **TextBlob NLP library** to classify reviews as positive, negative, or neutral. External APIs such as **weather and map services** are integrated to provide additional travel information.

The proposed system helps users make better travel decisions by combining intelligent itinerary generation with sentiment insights derived from travel reviews.
